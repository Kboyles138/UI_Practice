# UI Design System

A personal, reusable UI design system built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

## Components

Browse the sidebar to explore all components and their variants:

### Core

- **Button** — Primary, secondary, outline, ghost, danger. Sizes: sm, md, lg
- **Input** — Text input with label, error, hint
- **Card** — Elevated, outlined, filled. With Header, Title, Description, Content, Footer
- **Badge** — Status indicators: default, primary, success, warning, error, outline

### Selection Controls

- **Checkbox** — Single checkbox with label
- **RadioButton** — Radio group with options array
- **Toggle** — Switch/toggle with label
- **Select** — Native-style dropdown
- **MultiSelect** — Multi-option selection
- **Combobox** — Type-to-filter + select
- **Autocomplete** — Searchable input with suggestions

### Messages

- **Alert** — Inline alert with variants
- **Banner** — Full-width dismissible banner
- **Toast** — Floating notification
- **Snackbar** — Compact message with optional action
- **Notification** — Rich notification with title/timestamp

### Status

- **ProgressBar** — Progress indicator
- **Spinner** — Loading spinner
- **SkeletonLoader** — Content placeholder
- **LoadingIndicator** — Spinner with label

### Feedback

- **EmptyState** — Empty list/state placeholder
- **ErrorMessage** — Error text
- **SuccessMessage** — Success text
- **WarningMessage** — Warning text
- **InlineValidation** — Form field with validation message

### Media

- **Carousel** — Horizontal scroll with arrows
- **Image** — Image with Next.js optimization option
- **Avatar** — User avatar with fallback
- **Icon** — SVG icon wrapper
- **Logo** — Brand logo
- **LogoWithBrandName** — Logo + text

### Overlays

- **Modal** — Centered overlay
- **Dialog** — Confirmation dialog
- **Drawer** — Side panel
- **Popover** — Floating content
- **Tooltip** — Hover hint
- **Lightbox** — Full-screen image viewer

### Layout

- **Divider** — Horizontal/vertical rule
- **Separator** — Visual separator
- **Spacer** — Vertical spacing

### Date & Time

- **DatePicker** — Native date input
- **TimePicker** — Native time input
- **DateRangePicker** — Start and end date
- **Calendar** — Month grid date picker

### Data & Forms

- **Tabs** — Tabbed content
- **Table** — Data table with Head, Body, Row, Cell
- **FileUpload** — Drag-and-drop file upload
- **Dropdown** — Button with menu
- **RichTextEditor** — Basic rich text (bold, italic, underline)

## Addons

- **Docs** — Auto-generated documentation for each component
- **Controls** — Tweak props interactively
- **A11y** — Accessibility checks
- **Vitest** — Run component tests from stories
